Scripts for analyzing transcriptions of Hittite tablets from Sapinuwa, published at https://sapinuwatablets.org

- download.py: Downloads all PDFS from the Web site 
- decode.py: Extracts text from the PDFs into an HTML file
- unique.py: Creates list of previously unattested words in the tablets. To use, download the [TLHdig data set](https://zenodo.org/records/15459134) from Zenodo, unpack, and  pass the unpacked path as a command line argument to the script.
